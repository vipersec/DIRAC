===========================
dirac-wms-job-delete
===========================

  Delete DIRAC job from WMS, if running it will be killed

Usage::

  dirac-wms-job-delete [option|cfgfile] ... JobID ...

Arguments::

  JobID:    DIRAC Job ID 

Example::
 
  $ dirac-wms-job-delete 12
  Deleted job 12
