====================================
dirac-wms-job-get-output-data
====================================

  Retrieve the output data files of a DIRAC job

Usage::

  dirac-wms-job-get-output-data [option|cfgfile] ... JobID ...

Arguments::

  JobID:    DIRAC Job ID 

 

Options::

  -D:  --Dir=            : Store the output in this directory
