============================
dirac-wms-job-get-jdl
============================

  Retrieve the current JDL of a DIRAC job

Usage::

  dirac-wms-job-get-jdl [option|cfgfile] ... JobID ...

Arguments::

  JobID:    DIRAC Job ID
