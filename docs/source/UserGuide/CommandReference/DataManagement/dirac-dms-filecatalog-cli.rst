================================
dirac-dms-filecatalog-cli
================================

Launch the File Catalog shell

Usage::

   dirac-dms-filecatalog-cli [option]

 

 

Options::

  -f:  --file-catalog=   :    Catalog client type to use (default FileCatalog) 

Example::

  $ dirac-dms-filecatalog-cli
  Starting DIRAC FileCatalog client
