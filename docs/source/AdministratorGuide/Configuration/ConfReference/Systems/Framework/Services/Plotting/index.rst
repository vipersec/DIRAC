+-----------------+------------------------------------------+----------------------------+
| **Name**        | **Description**                          | **Example**                |
+-----------------+------------------------------------------+----------------------------+
| *PlotsLocation* | Path where data for monitoring is stored | PlotsLocation = data/plots |
