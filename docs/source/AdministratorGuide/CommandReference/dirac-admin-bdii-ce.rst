==========================
dirac-admin-bdii-ce
==========================

  Check info on BDII for CE

Usage::

  dirac-admin-bdii-ce [option|cfgfile] ... CE

Arguments::

  CE:       Name of the CE(ie: ce111.cern.ch) 

 

Options::

  -H:  --host=           : BDII host 

Example::

  $ dirac-admin-bdii-ce LCG.IN2P3.fr
