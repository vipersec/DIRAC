=================================
dirac-admin-bdii-ce-voview
=================================

  Check info on BDII for VO view of CE

Usage::

  dirac-admin-bdii-ce-voview [option|cfgfile] ... CE

Arguments::

  CE:       Name of the CE(ie: ce111.cern.ch) 

 

Options::

  -H:  --host=           : BDII host 

  -V:  --vo=             : vo 

Example::

  $ dirac-admin-bdii-ce-voview LCG.IN2P3.fr
