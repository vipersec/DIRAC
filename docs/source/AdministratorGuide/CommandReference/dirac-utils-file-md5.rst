===========================
dirac-utils-file-md5
===========================

Calculate md5 of the supplied file

Usage::

  dirac-utils-file-md5 [option|cfgfile] ... File ...

Arguments::

  File:     File Name 

Example::

  $ dirac-utils-file-md5 Example.tgz 
  Example.tgz 5C1A1102-EAFD-2CBA-25BD-0EFCCFC3623E
