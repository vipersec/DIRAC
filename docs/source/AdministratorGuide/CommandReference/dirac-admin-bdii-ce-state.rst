================================
dirac-admin-bdii-ce-state
================================

  Check info on BDII for CE state

Usage::

  dirac-admin-bdii-ce-state [option|cfgfile] ... CE

Arguments::

  CE:       Name of the CE(ie: ce111.cern.ch) 

 

Options::

  -H:  --host=           : BDII host 

  -V:  --vo=             : vo
