============================
dirac-admin-bdii-site
============================

  Check info on BDII for Site

Usage::

  dirac-admin-bdii-site [option|cfgfile] ... Site

Arguments::

  Site:     Name of the Site (ie: CERN-PROD) 

 

Options::

  -H:  --host=           : BDII host 

Example::

  $ dirac-admin-bdii-site CERN-PROD
