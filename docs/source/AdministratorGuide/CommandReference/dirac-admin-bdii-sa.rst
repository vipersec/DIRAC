==========================
dirac-admin-bdii-sa
==========================

  Check info on BDII for SA

Usage::

  dirac-admin-bdii-sa [option|cfgfile] ... Site

Arguments::

  Site:     Name of the Site (ie: CERN-PROD) 

 

Options::

  -H:  --host=           : BDII host 

  -V:  --vo=             : vo 

Example::

  $ dirac-admin-bdii-sa CERN-PROD
