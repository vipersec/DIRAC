==========================
dirac-admin-ce-info
==========================

  Retrieve Site Associated to a given CE

Usage::

  dirac-admin-ce-info [option|cfgfile] ... CE ...

Arguments::

  CE:       Name of the CE 

 

Options::

  -G:  --Grid=           : Define the Grid where to look (Default: LCG) 

Example::
  
  $ dirac-admin-ce-info LCG.IN2P3.fr
