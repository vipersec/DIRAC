===============================
dirac-admin-bdii-cluster
===============================

  Check info on BDII for Cluster

Usage::

  dirac-admin-bdii-cluster [option|cfgfile] ... CE

Arguments::

  CE:       Name of the CE(ie: ce111.cern.ch) 

 

Options::

  -H:  --host=           : BDII host 

Example::

  $ dirac-admin-bdii-cluster LCG.IN2P3.fr
